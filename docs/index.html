Quick Start

Clone and build from source:

git clone https://github.com/gmnstr/wav2amiga-web.git cd wav2amiga-web pnpm install pnpm build

Convert a single sample:

node apps/cli/dist/cli.js --mode single --note C-2 input.wav

Or use the web interface for drag-and-drop conversion.
